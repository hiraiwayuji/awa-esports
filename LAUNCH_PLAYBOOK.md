## 1. 立ち上げの根本思想

### 世界トップチームの共通点
- **勝つ前にカルチャーで人を集める**：FaZe ClanはYouTubeトリックショット動画から、100 ThievesはNadeshotのライフスタイルから始まった。先に「観たくなる集団」になり、後から競技的に強くなった。
- **メンバーが顔を出す**：選手はキャラクターであり、視聴者の友達。HIKO（Sentinels）、Sykkuno、Rachell（OfflineTV）など、人格そのものがコンテンツ。
- **クリエイター × アスリートの融合**：競技だけのチームは10年続かない。コンテンツも作れる選手を育てる/採る。

### AWA ESPORTSの独自ポジション
> **「徳島の物語ファクトリー」として機能する地域チーム。**

- 都市部メガチームのモノマネをしない
- 「徳島県民・元徳島限定」という制約をブランドに変える（=希少性）
- ufotable、阿波踊り、マチ★アソビと同じ系譜の「徳島が世界に出した文化」になる

### 3つの非交渉ライン（譲ってはいけない原則）
1. **嘘をつかない／盛らない**：勝率も応募数も、現実をそのまま発信する
2. **地元を踏み台にしない**：東京進出のための踏み台ではなく、徳島の中で完結する強さを優先
3. **メンバーの人生を毀損しない**：練習量、生活費、メンタル。長く続ける仕組みが最優先

## 2. ゲームタイトルの選定（戦略的フォーカス）

### 失敗の典型
> 「全タイトルやります」と宣言する → 練習も配信も中途半端 → どこにも勝てない、視聴者も伸びない

### 推奨タイトル：1次候補（2026年時点）

| タイトル | 推奨度 | 理由 |
| --- | --- | --- |
| **VALORANT** | ★★★★★ | 観戦人気、女性プレイヤー比率高、競技人口多、徳島でも対戦相手見つかる |
| **ストリートファイター6** | ★★★★ | EVO Japan等大会映え、ufotable連携アニメ親和性、シングル競技で運営楽 |
| **ぷよぷよeスポーツ** | ★★★★ | 全国eスポーツ高校選手権必修、シニア・学生どちらも参加可能 |
| **eFootball / FC25** | ★★★ | 自治体大会との接続、徳島の中学高校サッカー部からのリクルート可 |

### 2次候補（戦力余裕ができたら）
- League of Legends：競技シーン規模 No.1 だが運営コストも No.1
- Apex Legends：3人チーム制で身軽。配信映え抜群
- Rocket League：観戦体験◎、5名以下で回せる

### 選定の判断軸（毎四半期見直す）
1. **競技シーン規模**（賞金 / 大会数 / 観戦者数）
2. **観戦体験**（試合時間、ルール理解難易度）
3. **配信映え**（BGM、ハイライト適性）
4. **徳島文化との親和性**（コラボ可能性）
5. **アクセシビリティ**（年齢層、入門のしやすさ）

## 4. ガバナンス：3層構造と契約

### 3層
1. **Founders / 経営層**
   - 代表：ぼーるくん（最終意思決定 / 資金調達 / 対外）
   - 副代表（探す）：オペレーション担当、できれば財務に強い人
   - アドバイザリーボード：トーキンさん、長瀬教授、岩佐さん
2. **Coach / Staff 層**
   - ヘッドコーチ（タイトルごとに1人）
   - フィジカル担当：さみ整体（既決）
   - メンタル担当：未決（公認心理師or産業カウンセラーへ依頼）
   - 配信ディレクター：未決（コンソメさん中心に編成）
3. **Players 層**
   - スターター（試合出場）/ サブ / アカデミー（育成枠）の3階層

### 契約書（必須整備）

| 文書 | 用途 | 参考 |
| --- | --- | --- |
| **選手契約書** | 練習出席義務、賞金分配、肖像権、移籍ルール | プロ野球独立リーグ、Bリーグの選手契約をベース |
| **SNSガイドライン** | 発信NG行為、炎上時対応、案件発信ルール | YouTuber事務所のガイドラインを参考 |
| **賞金分配ルール** | 業界標準は **70:30〜60:40 in player favor**（チーム30〜40%） | T1, Cloud9のwriting |
| **退団 / 移籍** | 円満退団、契約期間、補償条項 | プロスポーツ選手契約 |
| **NDA** | スポンサー契約 / 戦略情報 | 標準IT企業NDA |

> **重要**：契約書は法律の素人で作らない。徳島弁護士会の若手弁護士に5万円程度で初稿レビューを依頼する。

### コミュニケーション運用
- **週次MTG**：月曜21時、Discord、30分。前週振り返り＋今週の練習計画
- **月次レビュー**：月初土曜、対面 or Zoom、KPI確認＋ランク発表
- **四半期戦略会議**：3ヶ月毎、全メンバー集合（徳島市内）

## 5. ランク制度の運用

> サイトの About にあるS/A/B/Cランクを、運用にどう落とすか。

### KPIシート（毎月集計）

| 指標 | 重み | 計測方法 |
| --- | --- | --- |
| 練習会出席率 | 20% | Discord打刻、月次集計 |
| 公式戦 / 大会成績 | 30% | 試合結果トラッキング |
| 配信時間 | 15% | Twitch / YouTube ダッシュボード |
| SNS発信回数 | 10% | X投稿、Instagram投稿 |
| チーム貢献 | 15% | 月次MTGでの相互投票 |
| 戦略議論への参加 | 10% | コーチによる定性評価 |

合計100点 → S+ (95-): / S (85-94) / A (70-84) / B (50-69) / C (49-)

### 昇降格の運用
- **四半期ごと**にメンバー一覧で公開発表（透明性が信頼を生む）
- **降格は事前面談**：本人の事情を聞く（家族、学業、仕事）
- **特典差**：
  - S+/S：公式ユニフォーム着用、大会出場優先、賞金分配率優遇
  - A：チームジャージ、定期戦出場
  - B：練習着、週次練習参加
  - C：見習い枠、月1練習見学＋オンラインスクリム参加

### よくある失敗
- 評価が成績偏重 → 配信が伸びない → スポンサー価値が出ない
- 評価が緩すぎる → 練習に来ない選手が残る → 真面目な選手が抜ける
- 評価が不透明 → 不公平感 → 内部対立

## 7. 練習体制の設計

### 練習の3レイヤー

| レイヤー | 頻度 | 場所 | 内容 |
| --- | --- | --- | --- |
| **個人練習** | 毎日 | 自宅 | 個人スキル、ランクマ |
| **チーム練習（オンライン）** | 週2 | Discord | スクリム、戦術確認 |
| **オフライン合宿** | 月1〜2 | 徳島市内 | チームビルディング、配信、撮影 |

### オフライン練習会（地域接続の最大の武器）
- **会場候補**：
  - 徳島大学・四国大学のメディアラボ（教授経由で借りる）
  - 徳島市内の貸会議室（駅前 / 徳島駅 K-2 ビル等）
  - スポンサー企業のオフィス（共催名目）
- **オープン参加**：第1〜第3週は内部練習、第4週は一般公開
- **観戦無料 + 入会案内ブース** → リクルートにつながる

### メンタル / フィジカル

| 項目 | 担当 | 頻度 |
| --- | --- | --- |
| 姿勢 / 肩こり / 腰痛ケア | さみ整体 | 月1集合チェック |
| 視力ケア（VDT症候群対策） | 徳島の眼科と連携 | 半年に1回 |
| 食事栄養指導 | 管理栄養士に依頼（要相談） | 季節ごと |
| メンタル相談 | 公認心理師（要採用） | 必要時、無料窓口 |

### 戦略レビュー
- **試合後 24h 以内**に動画レビュー会
- レビュー観点：構築、ポジショニング、コミュニケーション、メンタル
- 録画必須 → 後日アーカイブとして資産化

## 8. 収益モデル

> **重要**：1つの収益源に依存しない。最低5本の柱を持つ。

### 収益の5本柱

| 柱 | 想定収益（年） | 立ち上げコスト | 期間 |
| --- | --- | --- | --- |
| **1. スポンサー（地域活性 / トレーニング / 個人）** | ¥300〜600万 | ゼロ（HPあるだけ） | Y1から |
| **2. グッズ販売（アパレル）** | ¥100〜200万 | 在庫リスク¥30万 | Y1後半から |
| **3. ファンクラブ（月額）** | ¥50〜150万 | Discord/Patreon設定 | Y2から |
| **4. 大会賞金** | ¥0〜500万（変動大） | 出場費 | 不定期 |
| **5. 企業向け研修 / 出張授業** | ¥100〜300万 | 営業コスト | Y2から |

### スポンサー営業の進め方
1. **見込みリスト**を50社作る（徳島の元気な中小企業中心）
2. 1日1〜2社、ぼーるくん本人がアポ → 訪問
3. **資料は1枚** + Webサイトを見せる
4. 即決を狙わず、3回会って初めて契約
5. **最初の1社**は地域活性パートナー枠の半額でもOK（事例化が最大の資産）

### 営業先候補（参考）
- **食品 / 飲料**：地元菓子メーカー、酒造、徳島ラーメン店
- **住宅 / リフォーム**：地元工務店、不動産
- **クリニック / サロン**：歯科、整体、フィットネス（からだ鍼灸整骨院ぼーるくん人脈）
- **教育**：学習塾、英会話、IT教室（プログラミング、デザイン）
- **IT / 通信**：地域ISP、SI企業
- **アパレル / ファッション**：徳島ジーンズ、藍染メーカー（コラボのチャンス大）

### グッズ展開（在庫リスク最小から）
- **Phase 1**：Suzuri / BASE のオンデマンド受注生産（Tシャツ、ステッカー、トートバッグ）
- **Phase 2**：限定50枚のユニフォーム複製版
- **Phase 3**：藍染コラボ「徳島ジャージ」（地元工房と協業、希少性で高単価）

### ファンクラブ設計（参考：100 Thieves "100 Thieves Cash App Compound" / FaZe Subscriptions）
- **シルバー**：¥500/月 — 月1限定配信、Discord入会、ステッカー
- **ゴールド**：¥1,500/月 — 練習配信フル視聴、選手とのDiscord交流
- **プレミアム**：¥5,000/月 — 練習会VIP招待、選手と直接通話（年1）

## 9. 地域共生プログラム（徳島ならではの強み）

### 学校訪問プログラム（年6回ペース）
- **対象**：徳島県内 高校 / 大学 / 高専（徳島大、四国大、徳島文理大、阿南高専）
- **内容**：
  - e-sports体験会（VALORANT or SF6 を生徒対戦）
  - キャリアトーク（プロを目指すか、副業とするか、観戦者として楽しむか）
  - チーム選手のライブストリーミング
- **メリット**：
  - 学校側：「新しい部活動 / クラブ活動の選択肢」として歓迎される
  - チーム側：リクルートの最前線、保護者にも認知

### シニアe-sports教室（長瀬教授との連携）
- **背景**：四国大学が地域貢献研究で協力可能性大
- **内容**：60代以上向けの「ぷよぷよ」「太鼓の達人」などのカジュアルeスポーツ教室
- **意義**：認知症予防、地域コミュニティ、メディア露出（NHKに取り上げられる確率高）

### 商店街連動
- **阿波踊り期間（毎年8月）**：徳島駅前商店街に特設ブース、選手と触れ合えるイベント
- **個人・商店街サポーター枠**でフィーチャーされた店舗を、TikTokで「ぼーるくんが行く徳島の名店」として紹介

### 県人会連携（出ていった人の「徳島愛」を回収する）
- **関東徳島県人会**、**関西徳島県人会**にコンタクト
- 「元徳島県民」もメンバー応募できる仕組み（オンライン中心）
- 在京 / 在阪のサポーターとして月額500円〜の応援メンバー化

### マチ★アソビ / ufotable コラボの提案ライン
- **ステップ1**：マチ★アソビにブース出展（小規模イベント枠）
- **ステップ2**：ufotableのアニメ作品とのコラボストリーマー企画
- **ステップ3**：オリジナルアニメPVを制作（5年計画）

## 10. ブランド資産の育成

### ロゴ（最重要、岩佐さんに正式版発注）
- 暫定SVG（六角形 + "A"）→ 正式版へリプレース
- カラーバリエーション：藍 / ネオン / 単色 / モノクロ × フル / アイコンのみ
- 使用ガイドライン文書化

### ユニフォーム（藍染とネオン布のハイブリッド）
- **テクニカル要素**：ゲーミング向け吸湿速乾、長時間着座対応
- **デザイン要素**：徳島藍染を肩〜背中に、ネオン色をライン1本
- **限定50枚**：番号入り、本当に活動する選手のみ着用 → 希少性ブランディング

### アンセム楽曲（年1曲発表）
- 徳島の音楽家とコラボ（地元バンド、DJ、和楽器奏者）
- **使い道**：YouTubeイントロ、配信開始SE、入場曲
- **発表方法**：年1のチームムービーと同時公開

### ストーリーアセット（年1で蓄積）
- ドキュメンタリー長尺（30分）×1本
- 写真集 / フォトブック ×1冊（限定100部）
- 創業3年目に「3周年記念展示」を徳島駅前で開催

## 12. リスクとよくある失敗

### よくある失敗パターン

| 罠 | 兆候 | 対処 |
| --- | --- | --- |
| **代表が燃え尽きる** | ぼーるくんが全部やる、毎晩遅くまで作業 | 副代表 / オペレーション担当を3ヶ月以内に立てる |
| **メンバー間トラブル** | 練習に来ない、SNSで愚痴、契約違反 | 月次1on1、規約違反は即対応 |
| **タイトル散漫** | 全部やろうとして結果が出ない | 半年に1回タイトル絞り込みレビュー |
| **スポンサー依存** | 1社が大半を占める、その1社の機嫌伺い | 5本柱の収益、最大スポンサー比率は40%以下 |
| **配信放置** | 月1本も上がらない | コンテンツカレンダー固定、責任者を明確に |
| **法務無策** | 契約書なしで活動 → 後にトラブル | 設立3ヶ月以内に弁護士レビュー必須 |
| **税金 / 会計甘い** | 売上が出始めて慌てる | 開始時に税理士相談（顧問料月¥10,000程度） |

### クライシスマネジメント
- **選手が炎上**：24h 以内にチーム公式声明、必要なら活動停止 / 退団
- **大会で大敗** → 隠さず分析記事 → 「弱さを見せる強さ」がブランドになる
- **スポンサー突然撤退**：理由を学ぶ、他社で穴埋め、感情的にならない
